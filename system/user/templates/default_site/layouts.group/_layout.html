About me

I’m Steve McKinney, a web designer living in Manchester, England. I've been designing from young age and doing it ‘in industry’ for the past 4 years. My passion is with design, but I learnt to code out of necessity.

In all tutorials that I write I aim to provide understanding and rationale, as to why something is done a particular way. When designing and writing code, I believe this can help long term understanding. If there you have any feedback or need clarification, feel free to contact me.

You can find me elsewhere: dribbble, twitter, github and linkedin.
