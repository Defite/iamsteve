About me

My name is Steve McKinney, I run this website. For the majority of the time I’ve been involved in making websites, I've done it all. I started making websites around 2002, I was pretty young, so it was just a hobby to begin with. Generally, I saw no distinction between roles, like there is now, you designed and coded websites.

Now it’s much more complex, I'm first and foremost a designer, but I'm able to code my designs — 14 years does allow for that. There is a lot of debate around whether designers should code, I'm not here to push that, it’s whatever you want to do. It's not easy to learn and do both effectively, but you can certainly be a great designer and produce code. If you want to code your designs, this is what this website is about.
